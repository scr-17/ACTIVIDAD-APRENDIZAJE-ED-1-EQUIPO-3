# 📚 Actividad de Aprendizaje 1

## Estructura de Datos -- Listas, Pilas y Colas

------------------------------------------------------------------------

## 📌 Descripción

Este repositorio contiene la solución a la Actividad de Aprendizaje 1 de
la materia Estructura de Datos, cuyo objetivo es desarrollar programas
en lenguaje C utilizando estructuras lineales dinámicas:

-   Listas enlazadas
-   Pilas
-   Colas

Los programas implementan diferentes problemas prácticos aplicando
estructuras dinámicas y manejo de memoria.

------------------------------------------------------------------------

## 🎯 Competencia

El estudiante será capaz de:

-   Conocer las estructuras lineales.
-   Identificar cuándo utilizar listas, pilas o colas.
-   Aplicar estructuras dinámicas en la solución de problemas reales.
-   Implementar programas funcionales en lenguaje C.

------------------------------------------------------------------------

## 🛠️ Programas Desarrollados

### 1️⃣ Lista Enlazada de Números Enteros

**Funcionalidades:**

-   Crear una lista enlazada con números enteros positivos generados al
    azar.
-   Insertar los elementos al final de la lista.
-   Mostrar los elementos en pantalla.
-   Eliminar nodos que superen un valor ingresado por el usuario.

------------------------------------------------------------------------

### 2️⃣ Verificador de Frase Palíndroma

**Estrategia implementada:**

-   Se ingresa una frase.
-   Cada carácter se inserta en una pila y en una lista enlazada.
-   Se extraen simultáneamente los elementos.
-   Se comparan para determinar si es palíndromo.

------------------------------------------------------------------------

### 3️⃣ Cola Dinámica con Números Aleatorios

-   Se generan 100 números aleatorios en el rango -25 a +25.
-   Se almacenan en una cola dinámica.
-   Se genera una segunda cola con los números negativos.

------------------------------------------------------------------------

### 4️⃣ Mini-Directorio con Antivirus (ACME S.R.L.)

-   Implementación de lista enlazada con contactos (nombre y teléfono).
-   Simulación de virus que invierte nombres en posiciones múltiplos del
    día.
-   Función antivirus que corrige los nombres afectados.
-   Impresión de lista infectada y lista corregida.

------------------------------------------------------------------------

### 5️⃣ Desencriptador de Mensajes con Paréntesis

-   Uso de pilas para invertir contenido dentro de paréntesis.
-   Soporte para paréntesis anidados.
-   Reconstrucción del mensaje original.

------------------------------------------------------------------------

## 🧠 Requisitos Técnicos

-   Lenguaje: C
-   Uso de funciones
-   Uso de listas, pilas y colas
-   Manejo de memoria dinámica (malloc y free)
-   Compilación sin errores
-   Código documentado

------------------------------------------------------------------------

## ▶️ Compilación

En sistemas con GCC:

``` bash
gcc programa.c -o programa
./programa
```

En Windows:

``` bash
programa.exe
```
